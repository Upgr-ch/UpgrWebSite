3. Données collectées

Le site peut collecter les catégories de données suivantes :

  • Données d'identification : nom, adresse e-mail.
  • Données d'utilisation : logs, préférences, historique d'activité.
  • Données techniques : type d'appareil, version du système, adresse IP.
  • Données de navigation : cookies et données comportementales.

4. Finalités du traitement

Les données collectées servent à :

  • Assurer le bon fonctionnement et la sécurité du site.
  • Offrir des fonctionnalités personnalisées.
  • Analyser l'utilisation pour améliorer les services.
  • Répondre aux demandes de contact ou de devis.

5. Partage des données

Vos données ne sont partagées qu'avec :

  • Des prestataires techniques strictement nécessaires (hébergeur, outils d'analyse).
  • Les autorités compétentes, uniquement sur demande légale.

6. Durée de conservation

Les données sont conservées pendant la durée strictement nécessaire aux finalités pour lesquelles elles ont été collectées, dans le respect des obligations légales applicables.

7. Vos droits

Conformément à la LPD et au RGPD, vous disposez d'un droit d'accès, de rectification, d'effacement, de limitation et d'opposition au traitement de vos données. Pour exercer ces droits : kl@upgr.ch
